Step 1. Download the Repository

Download all project files and folders (app.py, tabs/, data/, assets/, requirements.txt, .gitignore).

Ensure that the folder structure remains unchanged.

-----------------------------------------------------------------------------------------------------------

Step 2. Create a Virtual Environment

In your IDE (e.g., VS Code, PyCharm) or terminal:

python -m venv ass2

-----------------------------------------------------------------------------------------------------------

Step 3. Activate the Virtual Environment

Windows (PowerShell):

ass2\Scripts\Activate.ps1


Mac/Linux:

source ass2/bin/activate

-----------------------------------------------------------------------------------------------------------

Step 4. Install Required Libraries

Once the environment is active, install dependencies listed in requirements.txt:

pip install -r requirements.txt

-----------------------------------------------------------------------------------------------------------

Step 5. Run the Dashboard

After installation, run the Streamlit app:

streamlit run app.py

-----------------------------------------------------------------------------------------------------------

Step 6. View the Dashboard

After running, a local URL will appear in your terminal, for example:

Local URL: http://localhost:8501

------------------------------------------------END-----------------------------------------------------------
